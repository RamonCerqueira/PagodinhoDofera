# Pagodinho do Fera

Landing page de alta conversão para contratação via WhatsApp com admin de agenda em Firebase.

## Stack
- Next.js 14 + TypeScript
- Tailwind CSS
- Framer Motion
- Firebase Firestore/Auth

## Setup
1. `npm install`
2. Crie `.env.local` com variáveis `NEXT_PUBLIC_FIREBASE_*`
3. `npm run dev`

## Rotas
- `/` site principal
- `/admin` painel CRUD de eventos
